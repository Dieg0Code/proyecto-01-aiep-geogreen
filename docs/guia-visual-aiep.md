# Guía visual interna AIEP para GeoGreen

> Uso interno para mantener consistencia en documentos, infografías,
> presentaciones y piezas de coordinación del proyecto GeoGreen Escolar Osorno.
> No reemplaza un manual oficial de marca. La regla práctica para este proyecto
> es: **usar vibe institucional AIEP sin insertar logos ni construir piezas que
> parezcan material corporativo oficial no validado**.

## Principio de uso

- Mantener una estética sobria, técnica y educativa.
- Usar rojo AIEP como acento principal, no como fondo dominante en todo el
  documento.
- Combinar rojo con azul profundo, grises fríos y fondos claros.
- Para GeoGreen, sumar verde solo como acento temático de sostenibilidad.
- No usar logos, isotipos ni versiones recreadas de marca en piezas internas.
- Identificar las piezas con texto simple: `AIEP Osorno`, `GeoGreen Escolar
  Osorno` o `Coordinación interna`, según corresponda.

## Paleta base

Los siguientes colores quedan como paleta operacional del proyecto. Están
alineados con los tokens ya usados en el sistema local de presentaciones
(`tools/slides-system/theme/tokens.js`) y con el aspecto institucional público de
AIEP.

| Uso | Color | HEX |
|---|---:|---|
| Rojo institucional/acento principal | Rojo AIEP | `#D62027` |
| Azul profundo para títulos, bandas o alto contraste | Azul profundo | `#102A43` |
| Texto principal | Tinta | `#243B53` |
| Texto secundario | Gris pizarra | `#52606D` |
| Líneas y bordes suaves | Borde cálido | `#D8CFC4` |
| Fondo cálido claro | Papel | `#F8F3EC` |
| Fondo neutro suave | Neutro suave | `#EDE6DA` |
| Fondo azul suave | Azul suave | `#E6EEF7` |
| Rojo suave para énfasis | Rojo pálido | `#F8E4E5` |
| Dorado para hitos/advertencias suaves | Dorado | `#E0BC5A` |
| Verde GeoGreen/acento ambiental | Verde sostenibilidad | `#3FAE6A` |
| Verde suave | Verde pálido | `#E9F6EE` |
| Blanco | Blanco | `#FFFFFF` |

## Reglas rápidas de composición

- Portadas: fondo claro, título grande en azul profundo o tinta, franja/acento
  rojo delgado y una bajada sobria.
- Documentos largos: encabezados con línea roja, páginas limpias, tablas con
  encabezado azul profundo.
- Infografías: usar rojo para jerarquía, verde para sostenibilidad, azul para
  estructura y grises para texto.
- Evitar piezas completamente rojas o completamente verdes.
- Evitar degradados llamativos, fondos recargados o estética publicitaria
  genérica.
- El contenido debe sentirse como material académico-profesional: claro,
  ordenado, respirado y fácil de revisar por docentes/equipo.

## Tipografía sugerida

- Documentos Office/PDF: Aptos o Aptos Display cuando esté disponible.
- Código o diagramas técnicos: Consolas.
- Evitar tipografías decorativas para documentos de coordinación.

## Aplicación a GeoGreen

GeoGreen puede usar verde como acento ambiental, pero la base visual debe seguir
siendo institucional:

- Títulos: azul profundo o tinta.
- Acentos, líneas y llamados: rojo AIEP.
- Sostenibilidad, sensores activos o resultados positivos: verde.
- Advertencias o hitos: dorado.
- Fondos: papel, blanco o neutro suave.
