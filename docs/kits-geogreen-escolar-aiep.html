Permite que cada solución tenga personalidad según el problema elegido.

No se espera un producto final industrial. Se espera que cada equipo pueda explicar una solución sostenible con lógica técnica básica: qué detecta, cómo decide, qué alerta o acciona, y qué valor aporta a la comunidad escolar.

Por qué UNO R4 WiFi

Es una placa Arduino actual, con microcontrolador principal a 5 V, USB-C, matriz LED y módulo WiFi/BLE integrado. Para taller reduce fricción frente a placas de 3.3 V cuando se usan sensores Arduino clásicos.

Precaución técnica

Aunque el UNO R4 WiFi integra un ESP32-S3 para conectividad, el trabajo de aula debe tratar la placa como Arduino UNO R4. Para módulos externos de 3.3 V o placas ESP32 sueltas, revisar niveles de voltaje antes de conectar.
