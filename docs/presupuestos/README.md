# Presupuestos y compras

Esta carpeta reúne documentos de compra, presupuesto y banco de componentes:

- Propuesta de kits y banco de componentes.
- Presupuesto de compra núcleo.
- Fuentes editables (`.html`, `.tex`).
- Versiones finales para compartir (`.pdf`).

Los valores son referenciales y deben validarse antes de compra institucional.
