El circuito por dentro

La caja negra que va bajo la tapa: un Arduino lee el sensor de distancia, convierte la lectura en porcentaje y enciende el semaforo. Al llenarse, el buzzer avisa.

HC-SR04 sensor ultrasonico - TRIG D9 / ECHO D10 Arduino UNO controlador LED verde D4 - nivel bajo LED amarillo D3 - nivel medio LED rojo D2 - lleno Buzzer D8 - alerta 3 x AAA alimentacion

Detalle real en arduino/wiring.yml y simulable en Wokwi.
