# Patrones de persuasión útiles

## Pertenencia

Usar cuando:
- conviene reforzar sentido de grupo.

Sirve para:
- que el estudiante se sienta parte de una cohorte, no un receptor pasivo.

## Progreso visible

Usar cuando:
- el grupo necesita notar avance.

Sirve para:
- mostrar que cada clase suma algo real y acumulativo.

## Relevancia profesional

Usar cuando:
- hay que conectar el contenido con el oficio técnico.

Sirve para:
- que lo aprendido no parezca tarea escolar vacía.

## Claridad de expectativa

Usar cuando:
- el grupo necesita saber exactamente qué hacer.

Sirve para:
- reducir fricción, excusas y desorden.

## Urgencia sana

Usar cuando:
- hay plazos o hitos cercanos.

Sirve para:
- mover a la acción sin sonar amenazante.

Evitar:
- dramatizar;
- culpar;
- exagerar consecuencias.

## Reconocimiento específico

Usar cuando:
- conviene reforzar conductas concretas.

Sirve para:
- premiar hábitos útiles como asistir, participar, preguntar, versionar o preparar material.

Evitar elogios vacíos como:
- `son secos`
- `cracks`
- `máquinas`

Preferir:
- reconocer una conducta concreta y por qué importa.
