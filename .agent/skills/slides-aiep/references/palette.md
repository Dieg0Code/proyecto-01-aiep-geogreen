# Paleta AIEP

## Colores base

- `AIEP Navy`: `#082B5C`
  Uso: portadas, aperturas de bloque, títulos fuertes, bandas estructurales y fondos institucionales.

- `AIEP Red`: `#D62027`
  Uso: acentos, chips, números clave, marcas de énfasis, pequeños divisores y llamados visuales.

- `Warm Light`: `#F5F2EC`
  Uso: fondo principal del deck cuando se necesite claridad y descanso visual.

- `White`: `#FFFFFF`
  Uso: superficies limpias, tarjetas, contraste interno y respiración.

- `Slate`: `#5F6B7A`
  Uso: texto secundario, subtítulos y apoyo visual de baja jerarquía.

- `Border`: `#D8DEE6`
  Uso: bordes, separadores, líneas suaves y cajas discretas.

## Colores de apoyo

- `Soft Blue`: `#E9EEF4`
  Uso: fondos suaves en slides conceptuales o comparativas.

- `Soft Neutral`: `#EEF3EF`
  Uso: superficies de descanso cuando convenga variar sin salir del tono institucional.

- `Warm Gold`: `#E0BC5A`
  Uso: acentos secundarios cálidos, divisores o chips de apoyo cuando el deck necesite más humanidad sin salir del tono institucional.

## Regla 60 / 30 / 10

- `60%`: fondos claros y superficies neutras.
- `30%`: azul institucional para estructura y peso visual.
- `10%`: rojo para acentos y énfasis.

## Reglas de uso

- El azul debe ser el color dominante cuando el deck necesite autoridad y cierre visual.
- El rojo se usa en dosis pequeñas; si aparece demasiado, la presentación pierde tono institucional.
- Los fondos claros deben sostener la legibilidad del contenido largo.
- Si se usa un color de apoyo, debe acompañar al azul y nunca competir con él.
- Mantener contraste alto entre texto y fondo.
